- ## import statement to use flask
  - install flask with pip(3)
    - ``` 
          pip3 install flask
      ```
  - import statement
    - ``` 
        from flask import Flask
      ```

- ## create a class from the Flash super-class
  - ``` 
      from flask import Flask
      app = Flask(__name__)

      [enter app code here]

      if __name__ == '__main__':
        app.run()
